# クラス

## オブジェクトが複数あるとき
```js:main.js
{
  const posts = [ 
    {
      text: 'JavaScriptの勉強中•••',
      likeCount: 0,
    }, 
    {
      text: 'プログラミング楽しい!',
      likeCount: 0,
    }, 
  ]

  function show(post) {
    console.log(`${post.text} - ${post.likeCount}いいね`); 
  }

  show(posts[0]);
}  
```
>JavaScriptの勉強中••• - 0いいね

```js:main.js
function show(post) {
    console.log(`${post.text} - ${post.likeCount}いいね`); 
  }
```
この処理をオブジェクトに直接書いて、データをまとめて管理することもできる
