``` 
<body>

```
HTMLファイルの中の<body></body>にいれる

```js:main.js
'use strict'; 
console.log('Hello World from main js!'); 
```


## 四則演算

```js:main.js
console.log(10 + 3); 
console.log(10 - 3); 
console.log(10 * 3); 
console.log(10 / 3); 
console.log(10 % 3); //あまり
console.log(10 ** 3); //累乗

console.log(2 + 10 *3); 
console.log((2 + 10) *3); 
```
