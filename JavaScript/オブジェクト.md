# オブジェクト

## オブジェクトとは

値に名前をつけて管理する。
```js:main.js
{
  // const point = [100, 180]; 

  // const point = {x: 100, y:180}; 

  const point = {
    x: 100, 
    y: 180, 
  }; 
  console.log(point); 
}
```
>{x: 100, y: 180}

オブジェクトを構成する要素ひとつひとつを __プロパティ(メンバー）__
左側を __名前（キー）__ 右側を __値__ と呼ぶ


## プロパティへのアクセス
```js:main.js
{
  const point = {
    x: 100, 
    y: 180, 
  }; 
  
  console.log(point.x); 
  console.log(point['y']);
}
```
>100

>100

`point.x`、`point['y']`の二通りで表せる。

### 値の変更
```js:main.js
{
  const point = {
    x: 100, 
    y: 180, 
  }; 
  
  point.x = 120; 
  // point['x'] = 120; 
}
```

### プロパティの追加と削除
```js:main.js
{
  const point = {
    x: 100, 
    y: 180, 
  }; 

  point.z = 90; 
  delete point.y; 
  console.log(point); 
}
```
>{x: 100, z: 90}


## オブジェクトの操作

### スプレッド構文

オブジェクトの中に別のオブジェクトを展開する
```js:main.js
{
  const otherProps = {
    r: 4, 
    color: 'red'
  };

  const point = {
    x: 100, 
    y: 180, 
    ...otherProps,　　　//スプレッド構文
  }; 

  console.log(point); 
}
```
>{x: 100, y: 180, r: 4, color: "red"}

### 分割代入とレスト構文

* __分割代入__ オブジェクトのプロパティのキーと同じ定数名を使って、値を代入する
* __レスト構文__ 残りのプロパティをオブジェクトのまま取り出したいとき
```js:main.js
{
  const point = {
    x: 100, 
    y: 180, 
    r: 4, 
    color: 'red'
  }; 
  
  const {x, r, ...others} = point;  //分割代入、レスト構文

  console.log(x);                   //定数としてアクセス
  console.log(r); 
  console.log(others);              //オブジェクトとしてアクセス
}
```
>100

>4

>{y: 180, color: "red"}


## Object.keysを使う
オブジェクトのプロパティを列挙するときに用いる
```js:main.js
{
  const point = {
    x: 100, 
    y: 180, 
  }; 

  const keys = Object.keys(point);                    //全てのキーを配列で取得できる
  keys.forEach(key => {                               //forEach()が使える
    console.log(`Key: ${key} Value: ${point[key]}`);  //文字列で取得されているので、point.keyではなくpoint[key]
  });
}
```
>Key: x Value: 100

>Key: y Value: 180


## 値へのアクセス
```js:main.js
{
  const points = [
    {x: 30, y: 20},   
    {x: 10, y: 50},   
    {x: 40, y: 40},   
  ];

  console.log(points[1].y); 
}
```
>50


## 変数の代入
簡単なデータ型と複雑なデータ型で挙動が異なる

#### 文字列
```js:main.js
{
  let x = 1; 
  let y = x; 
  x = 5; 
  console.log(x); 
  console.log(y);
}
```
>5

>1

#### 配列
```js:main.js
{ 
  let x = [1, 2]; 
  let y = x; 
  x[0] = 5; 
  console.log(x); //[5, 2]?
  console.log(y); //[1, 2]?
}
```
>[5, 2]

>[5, 2]

どちらも[5, 2]である。
