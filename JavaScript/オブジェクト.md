# オブジェクト

## オブジェクトとは

値に名前をつけて管理する。
```js:main.js
{
  // const point = [100, 180]; 

  // const point = {x: 100, y:180}; 

  const point = {
    x: 100, 
    y: 180, 
  }; 
  console.log(point); 
}
```
>{x: 100, y: 180}

オブジェクトを構成する要素ひとつひとつを __プロパティ(メンバー）__
左側を __名前（キー）__ 右側を __値__ と呼ぶ


## プロパティへのアクセス
```js:main.js
{
  const point = {
    x: 100, 
    y: 180, 
  }; 
  
  console.log(point.x); 
  console.log(point['y']);
}
```
>100

>100

`point.x`、`point['y']`の二通りで表せる。

### 値の変更
```js:main.js
{
  const point = {
    x: 100, 
    y: 180, 
  }; 
  
  point.x = 120; 
  // point['x'] = 120; 
}
```

### プロパティの追加と削除
```js:main.js
{
  const point = {
    x: 100, 
    y: 180, 
  }; 

  point.z = 90; 
  delete point.y; 
  console.log(point); 
}
```
>{x: 100, z: 90}


## オブジェクトの操作

### スプレッド構文

オブジェクトの中に別のオブジェクトを展開する
```js:main.js
{
  const otherProps = {
    r: 4, 
    color: 'red'
  };

  const point = {
    x: 100, 
    y: 180, 
    ...otherProps,　　　//スプレッド構文
  }; 

  console.log(point); 
}
```
>{x: 100, y: 180, r: 4, color: "red"}

### 分割代入とレスト構文

* __分割代入__ オブジェクトのプロパティのキーと同じ定数名を使って、値を代入する
* __レスト構文__ 残りのプロパティをオブジェクトのまま取り出したいとき
```js:main.js
{
  const point = {
    x: 100, 
    y: 180, 
    r: 4, 
    color: 'red'
  }; 
  
  const {x, r, ...others} = point;  //分割代入、レスト構文

  console.log(x);                   //定数としてアクセス
  console.log(r); 
  console.log(others);              //オブジェクトとしてアクセス
}
```
>100

>4

>{y: 180, color: "red"}


## Object.keysを使う
オブジェクトのプロパティを列挙するときに用いる
```js:main.js
{
  const point = {
    x: 100, 
    y: 180, 
  }; 

  const keys = Object.keys(point);                    //全てのキーを配列で取得できる
  keys.forEach(key => {                               //forEach()が使える
    console.log(`Key: ${key} Value: ${point[key]}`);  //文字列で取得されているので、point.keyではなくpoint[key]
  });
}
```
>Key: x Value: 100

>Key: y Value: 180


## 値へのアクセス

## 変数の代入
簡単なデータ型と複雑なデータ型で挙動が異なる

#### 文字列
```js:main.js
{
  let x = 1; 
  let y = x; 
  x = 5; 
  console.log(x); 
  console.log(y);
}
```
>5

>1

簡単なデータは値をコピーして保管される

#### 配列
```js:main.js
{ 
  let x = [1, 2]; 
  let y = x; 
  x[0] = 5; 
  console.log(x); //[5, 2]?
  console.log(y); //[1, 2]?
}
```
>[5, 2]

>[5, 2]

どちらも[5, 2]である。

複雑なデータは値を共有して保管される

#### スプレッド演算子
スプレッド演算子を使うことで、複雑なデータ型でも値をコピーして保管される
```js:main.js
{
  let x = [1, 2]; 
  let y = [...x]; 
  x[0] = 5; 
  console.log(x); //[5, 2]
  console.log(y); //[5, 2]
}
```
>[5, 2]

>[1, 2]


## 文字列を操作

### 文字列の文字数を表示
```js:main.js
{
  const str = 'hello'; 

  console.log(str.length); 
}
```
>5

### 文字列中の指定した文字を抽出
```js:main.js
{
  const str = 'hello'; 

  // str.substring(開始位置, 終了位置); 
  console.log( str.substring(2, 4)); 
}
```
>ll

### 指定した文字にアクセス
```js:main.js
{
  const str = 'hello'; 

  console.log(str[1]); 
  // str[1] = 'a'; ×
}
```
>e

### join() 
配列の要素を文字列として結合させる
```js:main.js
{
  const d = [2019, 11, 14]; 

  console.log(d.join('/'));
  console.log(d.join(''));
}
```
>2019/11/14

>20191114

### split()
文字列を分割して、配列に格納
```js:main.js
{
  const t = '17:08:24'; 
  
  console.log(t.split(':'));         //指定した引数で分割し、文字列として
  
  const [hour, minute, second] = t.split(':');　　//配列の要素を定数へ
  console.log(hour); 
  console.log(minute); 
  console.log(second); 
}
```
>["17", "08", "24"]

>17

>08

>24


## 数値を操作
```js:main.js
{
  const scores = [10, 3, 9]; 

  let sum = 0; 

  scores.forEach(score => {   //合計を求める
    sum += score;
  });

  const avg = sum / scores.length; 　//平均を求める

  console.log(sum); 
  console.log(avg); 

  console.log(Math.floor(avg)); //小数点以下切り捨て
  console.log(Math.ceil(avg));  //小数点以下切り上げ
  console.log(Math.round(avg)); //四捨五入
  console.log(avg.toFixed(3));  //指定した桁数にする

  console.log(Math.random());  //0以上1未満でランダムに数字を生成
}
```
>22

>7.333333333333333

>7

>8

>7

>7.333

>0.13608105335701515

### ランダムな整数値を作ってみよう
決められた範囲でランダムな整数値を出力する
```js:main.js
{
  // console.log(Math.random());

  // 0, 1. 2
  // Math.floor(Math.random() * 3)
  // 0, ....., n
  // Math.floor(Math.random() * (n+1))
  // min, ..., max
  // Math.floor(Math.random() * (max+1 - min)) + min

  console.log(Math.floor(Math.random() * 6) + 1); 
}
```
>5

```js:main.js
{
   // min, ..., max
   Math.floor(Math.random() * (max+1 - min)) + min
}
```
`(max+1 - min)`で範囲を設定する。
`+min`で範囲の開始位置を決める。


## 現在日時を扱う
```js:main.js
{
  const d = new Date(); 
  console.log(d); 

  console.log(`${d.getMonth() + 1} 月 ${d.getDate()} 日`); 
}
```
>Thu Apr 02 2020 23:35:34 GMT+0900 (日本標準時)

>4 月 2 日

例:年
```js:main.js
{
  const d = new Date(); 
  
  d.getFullYear();   2019
  d.getMonth();      0 - 11 (Jan:0 Feb:1) 
  d.getDate();       1 -31 
  d.getDay();        0 - 6  (Sun:0 Mon:1)
} 
```

例:時間
```js:main.js
{
  const d = new Date(); 

  d.getHours();     0 - 23
  d.getMinutes();   0 - 59 
  d.getSeconds();   0 - 59 
  d.getMilliseconds(); 0 - 999  1ms = 1/1000 sec
} 
```

### 特定の日時を使う
```js:main.js
{
  const d = new Date(2019, 10);  // 2019/11/01  00:00:00
  d.setHours(10, 20, 30);        // 2019/11/01  10:20:30
  d.setDate(31);                 // 2019/12/01  10:20:30
  d.setDate(d.getDate() + 3);    // 2019/12/04  10:20:30

  console.log(d);
}  
```


`new Date(2019, 10)`年と月が必須で、あとは省略すると自動で、日付は1、時間は0で作られる
